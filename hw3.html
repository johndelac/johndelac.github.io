3rd Assignment for CIS 114

Grades Calculator

    The button with the text "Grades Calculator" will navigate you to the page containing a script that will take valid input in order to calculate a students grade average. After computing the grade average, the script will return a letter grade.

Commission Calculator

    The button with the text "Commission Calculator" will navigate you to the page containing a script that will take valid input and will return how many items an employee has sold and the total weekly sales of said employee. After getting all those information, the script will return how much commission the employee will recieve on top of the $250 the employee receives weekly.

Temperature Converter

    The button with the text "Temperature Converter" will navigate you to the page containing a script that will take valid input and will return either the celcius conversion or fahrenheit conversion of the user's input.
